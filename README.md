
# SP_SwingTimer

Author: EinBaum

Swing timer bar for auto attacks. **Now for both hands!**

### Settings

Use these commands for configuration:
	/st
	/swingtimer

### SLAM MACRO.txt

This spammable macro will only cast Slam if the swing timer has just been reset after an auto attack.
